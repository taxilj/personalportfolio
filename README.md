# Taxil Prajapati | Personal Portfolio

A production-ready personal portfolio website built with Next.js 15, TypeScript, and Tailwind CSS. Inspired by the Dymas Alfin design style.

## Design Features
- **Clean Aesthetic**: White background with bold black condensed typography.
- **Cinematic Hero**: 3D depth effect with name layering and photo cutout.
- **Interactive Elements**: Custom cursor, smooth scroll reveals, and accordion services.
- **Responsive**: Fully optimized for mobile and desktop.
- **Accent**: Orange (#FF4D00) used for highlights and interactions.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Fonts**: Bebas Neue, Syne, DM Sans (via Google Fonts)
- **Icons**: Lucide React
- **Animations**: CSS Transitions + IntersectionObserver

## Setup Instructions
1. Clone the repository.
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure
- `/app`: Main application routes and layouts.
- `/components`: Reusable UI components (Hero, Navbar, SelectedWork, etc.).
- `/public`: Static assets (images, icons).
- `/lib`: Utility functions.
